What is Kyth?

Atomic. Gaming-ready. Yours.

Kyth is an immutable, atomic OS image — updates are applied as a whole, roll back if something breaks, and your system stays consistent. On top of that solid base, everything you need to game and work is already there.

🔒

Atomic & Immutable

The base system is read-only. Updates apply atomically and can be rolled back instantly if anything goes wrong.

🎮

Gaming Ready

Steam, Proton, and the tools you need are included from first boot. No driver hunting, no manual setup.

🛠️

Productivity First

Flatpak, development tools, and a stable base that won't break when you update. Work and game on the same machine.

🌐

Community Driven

Built by the community, for the community. Get help, share configs, and shape where Kyth goes next.
