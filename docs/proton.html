Proton (Steam Play) Quick Guide

A concise walkthrough to enable Steam Play and use Proton compatibility tools.

Prerequisites

  • A working Steam client (native or Flatpak)
  • Updated graphics drivers (Vulkan support recommended)

Enabling Steam Play

  1. Open Steam → Settings → Steam Play.
  2. Enable "Enable Steam Play for supported titles" and optionally "Enable Steam Play for all other titles".
  3. Choose a Proton version (Proton Experimental or a Proton-GE build).

Install Proton-GE (recommended)

Proton-GE provides community fixes for many titles. Install via ProtonUp-Qt or manually:

# example: install Proton-GE manually (download and extract)
mkdir -p ~/.steam/root/compatibilitytools.d
# extract the Proton-GE tarball into that directory
cp -r /path/to/Proton-GE-*.tar.gz ~/.steam/root/compatibilitytools.d/

Per-game override

  1. Right-click the game in Steam → Properties → Compatibility.
  2. Check "Force the use of a specific Steam Play compatibility tool" and select the Proton/Proton-GE build.

Troubleshooting

  • Verify Vulkan works: vulkaninfo or check drivers.
  • Try different Proton versions or Proton-GE builds for regressions.
  • Use the game's launch options for debug logs: PROTON_LOG=1 %command%
