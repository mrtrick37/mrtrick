Installation

Download the latest Kyth image (link coming soon). Use tools like Ventoy, Balena Etcher, or Fedora Media Writer to create a bootable USB. Boot from USB and follow the installer prompts.

Gaming Setup

Kyth comes with Steam pre-installed. Enable Steam Play for Proton compatibility, and use ProtonUp-Qt to install Proton-GE for additional fixes. Lutris and Heroic are available for non-Steam games.

Productivity Tools

Office, development, and creative tools are available via Flatpak and the built-in package manager. See the guides for recommended apps and setup tips.

Troubleshooting

For help with drivers, updates, or gaming issues, check the community forums or Discord. See the docs for common fixes and tips.
