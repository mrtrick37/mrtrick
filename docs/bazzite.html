Bazzite Setup Guide

Complete guide to installing, configuring, and optimizing Bazzite for a polished Linux gaming setup.

Download Bazzite Official Docs

What is Bazzite?

Bazzite is an OCI image based on Fedora Silverblue designed for gaming-first workflows. It uses immutable updates, ships key gaming tools out of the box, and supports desktop and handheld use cases.

  • Immutable + atomic updates for stability and easy rollback
  • Steam, Lutris, and Heroic available immediately
  • HDR/VRR support and modern gaming stack integrations
  • Good hardware support including Nvidia variants

Quick Install Path

  1. Download: Grab the correct image from bazzite.gg based on device, GPU, and desktop environment.
  2. Create USB: Flash ISO using Ventoy, Balena Etcher, or Rufus.
  3. Install: Boot USB, run installer, set user/timezone/partitioning.
  4. First boot: Complete setup wizard and apply available updates.
  5. Start gaming: Sign into Steam and validate compatibility on ProtonDB.

Installation Media Tools

Ventoy

Boot multiple ISO files from one USB drive.

 Balena Etcher

Beginner-friendly cross-platform flashing utility.

 Rufus

Fast, reliable USB creator for Windows systems.

Gaming Tool Setup

Steam

  1. Open Steam and sign in.
  2. Enable Steam Play for all titles in Steam settings.
  3. Use ProtonDB entries when a game needs launch tweaks.

Additional Launchers

  • Lutris: Manage Epic, GOG, and other launchers.
  • Heroic: Streamlined Epic and GOG game access.
  • Bottles: Optional Windows app compatibility tool.

Troubleshooting

Game Won't Launch

  • Verify files in Steam.
  • Try another Proton version per title.
  • Check known fixes on ProtonDB.

Poor Performance

  • Confirm power profile/GameMode behavior.
  • Validate GPU driver detection.
  • Tune graphics settings and upscaling options.

Audio Issues

  • Check PipeWire: systemctl --user status pipewire
  • Restart PipeWire: systemctl --user restart pipewire
