Quick Start Guides

Four distro paths, one goal: install quickly and get gaming with less setup friction.

Bazzite

Immutable gaming-focused Fedora image

  1. Pick image: Download your hardware image from bazzite.gg (desktop, handheld, or Nvidia variants).
  2. Create USB: Flash with Ventoy, Balena Etcher, or Rufus and verify the write succeeded.
  3. Install: Boot live media, run installer, set disk layout, timezone, and user account.
  4. First boot: Update system, enable game mode/power profile, and check GPU driver status.
  5. Gaming apps: Sign into Steam, then add Heroic/Lutris for non-Steam libraries if needed.
  6. Validate games: Check compatibility in ProtonDB and apply suggested launch options.
Download Bazzite Bazzite Docs

CachyOS

Performance-tuned Arch-based distro

  1. Download ISO: Get the latest release from cachyos.org.
  2. Prepare USB: Flash and boot live environment with UEFI enabled.
  3. Calamares setup: Choose desktop, kernel profile, and partitioning strategy.
  4. Post-install sync: Run full update and mirror refresh before installing games.
  5. Gaming stack: Install Steam, Proton-GE, and optional tools like MangoHud/GameMode.
  6. Tune + test: Validate FPS baseline in one game, then apply kernel/governor tweaks.
Download CachyOS CachyOS Wiki

Aurora

Stable immutable workstation image

  1. Select image: Use getaurora.dev to pick the correct hardware variant.
  2. Create installer: Build USB with Fedora Media Writer (recommended by project docs).
  3. Install system: Complete guided install and reboot into your new immutable base.
  4. Initial setup: Run first updates and configure Flatpak/software sources.
  5. Gaming tools: Install Steam and helper tools, then sign into game launchers.
  6. Verify workflow: Test one game, check controller/audio/display behavior, and snapshot settings.
Download Aurora Aurora Docs

Nobara

Fedora-based distro tuned for gaming

  1. Download ISO: Get the latest release from nobaraproject.org.
  2. Create USB: Write the image with Ventoy, Balena Etcher, or Fedora Media Writer.
  3. Install: Boot the live environment, run installer, and complete user/timezone/storage setup.
  4. First update: Update packages and reboot once to ensure drivers and kernel are current.
  5. Gaming setup: Install Steam, enable Steam Play, and optionally add ProtonUp-Qt for Proton-GE.
  6. Validate titles: Test one game and check fixes on ProtonDB if needed.
