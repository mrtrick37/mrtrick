Quick Start Guide

Pick your distro and follow the quick steps to install and start gaming fast.

Bazzite

  1. Download the correct ISO from bazzite.gg.
  2. Create a bootable USB with Ventoy, Balena Etcher, or Rufus.
  3. Boot, run installer, and complete first-time setup.
  4. Sign into Steam and verify games on ProtonDB.
Open Bazzite

CachyOS

  1. Download ISO from cachyos.org.
  2. Write ISO to USB and boot the live environment.
  3. Install via Calamares and choose your desktop/kernel preset.
  4. Update system, install Steam/Lutris, and enable gaming tweaks.
Open CachyOS

Aurora

  1. Go to getaurora.dev and pick your hardware image.
  2. Create installation USB using Fedora Media Writer.
  3. Install Aurora and reboot into the configured desktop.
  4. Open software store, install apps, and connect game libraries.
Open Aurora
