What is Kyth?

Atomic. Gaming-ready. Yours.

An immutable OS image where updates apply atomically and roll back if something breaks. On top of that stable base, everything you need to game and work is already included.

🔒

Atomic & Immutable

The base system is read-only. Updates apply atomically and can be rolled back instantly if anything goes wrong — no more broken systems after an update.

🎮

Gaming Ready

Steam, Proton, and the tools you need are included from first boot. No driver hunting, no manual configuration — just install and play.

🛠️

Productivity First

Flatpak, development tools, and a stable base that won't break when you update. Work and game on the same machine without compromise.

🌐

Community Driven

Built by the community, for the community. Get help, share configs, and shape where Kyth goes next — on Discord or in the forums.
